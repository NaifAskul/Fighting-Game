2. clicking on index.html

     b. audio folder (includes sound effects)
     c. img folder (includes Frame-By-Frame Animations and textures)
     d. js folder (includes classes and utilities needed for the game to work)

   B. Can work without internet


Enjoy :)

Note: Make sure to have the keyboard language on English not Arabic in order to use left player.
